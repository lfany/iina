Terminologies

  • "left" and "right" button: the rewind / fast forward button to the left and right of play button

Checklist

  • General
  • Launchable:
    • No crach during launch
    • Able to show player window when opened a file
    • Able to show preference window when choose IINA - Preference in menu or Command+,
  • Decoding:
    • Able to play all common video types
    • Refuse to open unrecognized files
  • Theme:
    • All 4 themes can be changed in settings and works
  • App behaviour:
    • All settings in Preference - General - Behaviour works without need of restarting
  • Screenshots:
    • Screenshot menu item and shortcut works
    • Able to change screenshot folder
    • Able to change screenshot format
    • Able to set whether include subtitles
    • Able to go to screenshot folder in menu
  • UI
    • Window:
      • Any action should not make window smaller than 500*300
      • It should adjust to video size (with animation) when opened file
      • When "use physical resolution on retina screen" in preference checked, it should resize window based on actual resolution
      • When "resize window only when open file manually" in preference checked, it will remain the same width when switching among playlist items
      • All size related actions in menu Video should work; "double size" should not make window bigger than screen
      • Able to set and cancel "Always on top" in menu Video.
    • On screen control
      • It should contain a mute button, a volume slider, a quick settings button and a playlist button
      • It should display video length, current positon and a progress bar
      • Dragging osc should be possible, and its frame should be bounded to window
      • When enabled in preference, it will stick to center when dragging
      • It's possible to set a hide duration for osc. After such duration without any mouse operation, the titlebar, osc and cursor will be hidden.
    • On screen display
      • Osd should be shown when certain property changed. Currently only some properties are observed, please check OSDMessage.swift for a list.
      • It's possible to set text size of osd
      • It's possible to set a hide duration for osd
    • Quick settings panel
      • It can be triggered by clicking quick settings button in osc, or from menu (video, audio, subtitle - quick settings panel). If open from menu, it should appear with corresponding tab selected, otherwise video tab is selected by default.
      • Click quick settings btn again, run menu actions again or single click anywhere else in the window should hide quick settings panel
    • Playlist panel
      • Basically the same as quick settings panel
    • Fullscreen
      • Able to toggle fullscreen in menu Video and standard (green traffic light) button
      • When in fullscreen, it should have a normal titlebar and support all macOS standard actions
      • Video view should be put in center and best fit the screen size.
  • Playback
    • Pause:
      • Able to pause through menu and clicking play button
      • "Stop and clear playlists" in menu works
    • Seek:
      • Able to seek exactly when change the progress slider
      • When mouse hovered progress slider, a preview label should be displayed above cursor, and after seeking, the playback position should be exactly the time shown in the label
      • Step forward / backward actions in menu should work, and so does frame step (pressing option)
      • Able to jump to beginning and jump to any time in menu; invalid input in "jump to" should trigger alert
    • A-B Loop
      • A-B loop related actions in menu should works
  • Video
    • (Syncronized means the sliders / input boxes in quick settings panel should display the current value when opened.)
    • Track:
      • Able to show all video tracks in a file
      • Able to switch between video tracks through quick settings and menu
      • Current track in menu and quick settings should be syncronized
    • Aspect ratio:
      • Able to set common aspect ratios in quick settings and menu
      • Able to input custom aspect ratios in quick settings
      • Aspect ratio in menu and quick settings should be syncronized
    • Crop:
      • Able to set common croppoing aspect ratios in quick settings and menu
      • Able to enter interactive crop mode by clicking "Custom" button in quick settings
        • Able to drag the edges of the box to change crop region
        • The crop region should not be bigger than video size
        • Able to select common cropping aspect ratios, and crop region should change respectively
        • Able to set the cropping by clicking "done" button
        • Able to expand to original video size after cropping
        • Crop status in menu and quick settings should be syncronized
    • Rotation:
      • Able to set rotation in quick settings and menu
      • Rotation status in menu and quick settings should be syncronized
    • Flip:
      • Able to set and cancel flip / mirror in menu
    • Speed:
      • Able to set speed in quick settings
      • Able to set custom speed value in quick settings
      • Able to set multiple level of speed when clicking left and right button (when chosen "use left/right btn for speed" in preference)
      • Speed should be reset when video paused
    • Deinterlace:
      • Able to set deinterlace in quick settings and menu
      • Deinterlace status in menu and quick settings should be syncronized
    • Equalizer:
      • Able to adjust equalizers in quick settings view
      • Aboe to reset them by clicking reset button
    • Video filters:
      • Able to inspect all current video filters in video filters window
      • Able to add new video filters (in mpv's vf format)
      • Able to edit existing filter
      • Able to delete video filters
      • Show warning if input invalid vf string
      • Filters added by IINA (e.g. flip, crop) can be edited or deleted, and status should be syncronized with quick settings view
    • Codec: Codec settings in preference - codec should work
  • Audio
    • Track: Basically the same with video track.
    • External audio: Able to load external audio track from menu and quick settings
    • Delay: Same as above requirement
    • Equalizer: Able to set equalizer and reset
    • Audio filters: Basically the same with video filters
    • Codec: Codec settings in preference - codec should work
  • Subtitles
  • Control
    • A single click will only hide the quick settings panel if its opened
  • Key bindings
